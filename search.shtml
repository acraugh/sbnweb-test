SBN Search

The Small Bodies Node of NASA's Planetary Data System is devoted to archiving, cataloging, and distributing scientific data sets concerning small bodies of the Solar System (asteroids, comets, interplanetary dust).

The SBN datasets are available on-line for browsing and downloading. Each data set is a collection of documentation (text or pdf) and data files, the majority of which are in the form of ASCII or binary format. Detailed descriptions of the various file types you might encounter are available.
