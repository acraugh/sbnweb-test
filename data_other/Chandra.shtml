Catalog of Chandra X-ray Telescope Small Bodies Observations

This page provides links to Small Bodies related observations made with the Chandra X-ray Telescope. The SBN does not archive any data from Chandra, but this page may assist scientists in finding the desired observations in the Chandra Data Archive. This listing is, as far as we know, complete through August 2019. The list is updated periodically. Click here to search for the most up-to-date list of SOLAR SYSTEM observations from the source.

The tables below each contain a list of related objects, their classification, and the number of their Chandra observations. Clicking on an object's name will bring you to a page for that object, listing each observation and providing important data to help you access the data using Chandra's observation search engine.

Because of the number of objects, the data have been broken up into different categories, seen by the links below. Feel free to use these links to get to the catagory that you are looking for, but note that some of these lists may be huge. It is recommended that you do a search within the web browser to find the object you are looking for. In most web browsers, to do this you will need to press CTRL-F and type part of the name or designation of the object.
